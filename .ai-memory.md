# AI Memory — Service Manager (Electron)

## Tech Stack
- Electron 28, Node.js, vanilla JS (no bundler)
- nodeIntegration: true, contextIsolation: false (renderer dùng require trực tiếp)
- electron-builder (NSIS target, Windows)

## Project Structure (sau refactor)
```
src/
  main/
    index.js       ← entry point Electron (createWindow, lifecycle, shortcuts)
    ipc.js         ← tất cả ipcMain.on handlers; SERVICE_CONFIGS để thêm service mới
    services.js    ← spawn/stop/update/getStatus; processes registry
    settings.js    ← load/save settings.json từ userData
    tray.js        ← Tray create/updateMenu/notify
  renderer/
    index.js       ← entry point renderer (init tất cả modules)
    state.js       ← runtime state { router, openclaw }
    ui.js          ← DOM helpers: setStatus, addLog, toast, metrics, alert, ripple...
    views/
      dashboard.js ← service cards, IPC handlers cho status/log/update
      logs.js      ← clear all logs
      settings.js  ← load/save settings UI
    components/
      logPanel.js       ← collapsible log + filter
      commandPalette.js ← Ctrl+K palette, extensible qua addCommand()
index.html         ← trỏ src/renderer/index.js
package.json       ← main: src/main/index.js
```

## Patterns
- SERVICE_CONFIGS trong ipc.js: thêm service mới chỉ cần thêm 1 entry
- commandPalette.addCommand() để đăng ký lệnh mới từ module khác
- Mỗi view là module độc lập export { init }
- ui.js là pure DOM helpers, không có state
- **Renderer require path**: index.html dùng `require(path.join(__dirname, 'src/renderer/index.js'))` để __dirname trong các module con resolve đúng
- **Settings**: settings object có `_save()` và `_settingsPath` (không serialize vào JSON), chỉ save các key trong DEFAULTS + windowBounds

## Services hiện tại
- 9Router: port 20128, cmd `9router`, npm pkg `9router`
- OpenClaw: port 18789, cmd `openclaw gateway`, npm pkg `openclaw`

## Settings path
- `%APPDATA%\claw-router-manager\settings.json`

## Follow-ups / Mở rộng
- Thêm service mới: thêm entry vào SERVICE_CONFIGS (ipc.js) + card HTML + state entry
- Thêm view mới: tạo src/renderer/views/myview.js, import trong renderer/index.js
- Thêm tiện ích máy tính: tạo src/renderer/views/tools/ và src/main/tools/

## Tools đã có
- **Hẹn giờ tắt máy** (`src/renderer/views/tools/shutdown.js` + `src/main/tools/shutdown.js`)
